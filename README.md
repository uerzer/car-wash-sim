# Car Wash Sim

A browser-based car wash simulation game built with vanilla JavaScript.

## About

Interactive car wash simulation. Drive cars through the wash, manage the queue, and keep customers happy.

## Tech Stack

- Vanilla JavaScript
- HTML5 / CSS3
- Netlify deployment

## Live Demo

Deployed on Netlify.

## Development

Open `index.html` directly in a browser or use a local dev server:

```bash
npx serve .
```
